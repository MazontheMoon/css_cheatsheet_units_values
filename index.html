CSS Values and Units

Numeric Values

Numeric values are used to set length and size of elements.

Absolute values always remain the set size regardless of any other related settings

  • px
  • mm
  • cm
  • in
  • pt (points = 1/72 inch)
  • pc (pica = 12 points)

Relative values are relative to the current elements font-size or viewport size

  • em (M)
  • ex (x - not common)
  • ch (0 - not common)
  • rem (root em)
  • vw (1/100th width of viewport
  • vh (1/100th height of viewport

Unitless Values

Unitless values are most commonly used to remove values, such as margin:0;

Percentages

Can be used to specify most things that can be specified with specific numerical values. This allows you to create, for example, boxes whose width will always shift to be a certain percentage of their parent containers width.

Unitless values are most commonly used to remove values, such as (font-size: 200% = 2em = 32px)

Colour Values

  • Keywords

    165 values

    background-color: red;

  • Hexadecimal

    256 values

    background-color: #ff0000;

  • RGB

    Red Green Blue

    256 values

    background-color: rgb (0,0,255);

  • HSL

    Hue Saturation Light

    16.7 million values

    background-color: hsl(240, 100%, 50%);

  • RGBA

    Red Green Blue Alpha (transparency)

    background-color: rgba (255,0,0,0.5);

  • HSLA

    Hue Saturation Light Alpha (transparency)

    background-color: hsla(240, 100%, 50%, 0.5);

  • Opacity

    Opacity can also be set using the opacity property but it is applied to all selected elements and their children

    {background-color: rgb (255,0,0);opacity: 0.5;}
